# Dossier d'architecture

## Présentation

Le dossier d'architecture contient la documentation détaillée de l'architecture technique d'Helios.

## Contenu

### Architecture générale

- Découplage datacrawler / backend / frontend
- Clean Architecture
- Flux de données

### Composants

- **Datacrawler** : récupération, transformation et chargement des données
- **Backend** : API et logique métier
- **Frontend** : interface utilisateur Next.js/React
- **Base de données** : schéma et relations

### Sécurité

- Authentification et autorisation
- Chiffrement des données
- Gestion des secrets

### Performance

- Optimisation des requêtes
- Cache
- Index de base de données

## Localisation

Le dossier d'architecture est disponible dans le dépôt de code ou dans un espace de documentation dédié.
