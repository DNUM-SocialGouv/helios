# Paramétrage de l'aide

## Présentation

Le paramétrage de l'aide permet aux administrateurs de gérer le contenu de la section Aide accessible aux utilisateurs.

## Actions possibles

### Création d'un contenu d'aide

[Capture écran : création de contenu]

1. Accéder au paramétrage de l'aide depuis le menu Administration
2. Cliquer sur "Créer un nouveau contenu"
3. Définir :
   - Le titre du contenu
   - Le type de contenu (FAQ, Guide, Aide contextuelle)
   - La section d'affichage
   - Le contenu (texte, images, liens)
   - La page associée (si aide contextuelle)
4. Sauvegarder

### Modification d'un contenu existant

1. Accéder au paramétrage de l'aide
2. Repérer le contenu à modifier dans la liste
3. Cliquer sur "Modifier"
4. Apporter les modifications nécessaires
5. Sauvegarder

### Suppression d'un contenu

1. Accéder au paramétrage de l'aide
2. Repérer le contenu à supprimer
3. Cliquer sur "Supprimer"
4. Confirmer la suppression

## Types de contenus

### FAQ

Questions fréquentes avec réponses. Organisées par thème.

### Guide d'utilisation

Tutoriels et guides pas à pas pour utiliser les fonctionnalités.

### Aide contextuelle

Aide spécifique à une page ou une fonctionnalité, accessible via une icône d'aide.

### Informations de contact

Coordonnées et formulaire de contact pour le support.

## Règles

- Les administrateurs nationaux peuvent modifier tous les contenus
- Les administrateurs régionaux peuvent modifier les contenus de leur région uniquement
- Les modifications sont immédiatement visibles par les utilisateurs
- Les contenus peuvent inclure du texte formaté, des images et des liens

## Organisation

Les contenus peuvent être organisés en sections et sous-sections pour faciliter la navigation des utilisateurs.
