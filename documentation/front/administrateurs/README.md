# Administrateurs

Cette section documente les fonctionnalités réservées aux administrateurs (ADMIN_NAT et ADMIN_REG).

## Fonctionnalités d'administration

- **[Paramétrage des autorisations](parametrage-autorisations.md)** : définition des indicateurs et blocs visibles par profil métier
- **[Paramétrage de l'aide](parametrage-aide.md)** : gestion du contenu d'aide (création, modification, suppression)
- **[Console d'administration](console-administration.md)** : gestion des utilisateurs, comptes et supervision

## Distinction administrateur national / régional

- **Administrateur national (ADMIN_NAT)** : accès à toutes les régions, gestion de tous les comptes, paramétrage global
- **Administrateur régional (ADMIN_REG)** : accès limité à sa région, gestion des comptes de sa région uniquement

## Accès

[Capture écran : menu Administration]

Les fonctionnalités d'administration sont accessibles depuis le menu "Administration" (visible uniquement pour les rôles ADMIN_NAT et ADMIN_REG).
