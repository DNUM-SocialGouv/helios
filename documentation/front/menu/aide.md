# Aide

## Points d'entrée disponibles

[Capture écran : points d'entrée aide]

L'aide est accessible depuis plusieurs points d'entrée :

1. **Menu principal** : lien "Aide" dans le menu de navigation
2. **Icône d'aide contextuelle** : présente sur certaines pages pour une aide spécifique
3. **Bas de page** : liens "FAQ" et "Nous contacter" dans le pied de page

## Contenu de l'aide

[Capture écran : page Aide]

Le contenu de l'aide est **paramétrable** par les administrateurs (voir [Paramétrage Aide](../administrateurs/parametrage-aide.md)).

### Sections disponibles

- **FAQ** (Foire Aux Questions) : questions fréquentes et réponses
- **Guide d'utilisation** : tutoriels et guides pas à pas
- **Nous contacter** : formulaire de contact ou informations de contact
- **Aide contextuelle** : informations spécifiques selon la page consultée

### Contenu paramétrable

Les administrateurs peuvent :
- Créer, modifier ou supprimer des sections d'aide
- Ajouter des contenus spécifiques par page
- Mettre à jour la FAQ
- Modifier les informations de contact

## FAQ

[Capture écran : section FAQ]

La FAQ contient des questions fréquentes organisées par thème :

- **Connexion et compte** : création de compte, mot de passe oublié, etc.
- **Recherche** : comment rechercher, utiliser les filtres, etc.
- **Fiches** : comprendre les indicateurs, navigation, etc.
- **Comparaison** : comment comparer des établissements
- **Listes** : création, gestion, import de listes
- **Administration** : pour les administrateurs

## Nous contacter

[Capture écran : formulaire Nous contacter]

La section "Nous contacter" permet de :

- Poser une question
- Signaler un problème
- Faire une demande d'assistance
- Proposer une amélioration

Le formulaire peut être configuré pour envoyer les messages à différentes adresses selon le type de demande.

## Aide contextuelle

[Capture écran : icône aide contextuelle]

Sur certaines pages, une icône d'aide (?) est présente à côté d'éléments spécifiques :

- Explication d'un indicateur
- Aide sur une fonctionnalité
- Règles métier associées

En cliquant sur l'icône, une infobulle ou une page d'aide s'ouvre avec des informations détaillées.

## Évolution du contenu

Le contenu de l'aide peut évoluer régulièrement selon :
- Les retours utilisateurs
- Les nouvelles fonctionnalités
- Les mises à jour du produit

Les administrateurs sont responsables de la mise à jour du contenu (voir [Paramétrage Aide](../administrateurs/parametrage-aide.md)).
