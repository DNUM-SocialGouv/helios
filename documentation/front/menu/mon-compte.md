# Mon compte

## Accès

[Capture écran : menu Mon compte]

Le menu "Mon compte" est accessible depuis le menu utilisateur en haut à droite de l'interface.

## Informations affichées

[Capture écran : page Mon compte]

La page "Mon compte" affiche les informations suivantes (non modifiables par l'utilisateur) :

- **Nom** : nom de famille
- **Prénom**
- **Email** : adresse email de connexion
- **Institution** : institution de rattachement (région, administration centrale, etc.)
- **Rôle** : rôle attribué (USER, ADMIN_REG, ADMIN_NAT, ADMIN_CENTR)
- **Profils métiers** : liste des profils métiers associés (financier, RH, qualité, activité, etc.)
- **Date de création du compte**
- **Date de dernière connexion**

## Modification des informations

Les informations affichées dans "Mon compte" ne sont **pas modifiables** par l'utilisateur lui-même.

Pour modifier ces informations, il faut contacter :
- Un **administrateur régional** pour les utilisateurs de sa région
- Un **administrateur national** pour toutes les modifications

## Actions disponibles

Depuis la page "Mon compte", l'utilisateur peut :

- **Changer son mot de passe** : accès au formulaire de changement de mot de passe
- **Consulter l'historique** : accès à l'historique de ses recherches
- **Voir ses listes** : accès à "Mes listes"
