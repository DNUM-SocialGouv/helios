# Historique

## Présentation

L'historique conserve la liste des fiches consultées par l'utilisateur, permettant de retrouver rapidement des établissements ou entités juridiques récemment consultés.

## Accès

[Capture écran : menu Historique]

L'historique est accessible depuis :
- Le menu principal : "Historique"
- Le menu utilisateur : "Mon compte" puis "Historique"

## Contenu de l'historique

[Capture écran : page Historique]

L'historique affiche la liste des fiches consultées avec :

- **Raison sociale** de l'établissement ou de l'entité juridique
- **Numéro FINESS**
- **Type** (sanitaire, médico-social, entité juridique)
- **Date et heure de consultation**
- **Origine de la consultation** :
  - Depuis une recherche
  - Depuis une comparaison
  - Depuis un nouvel onglet (ouverture directe)

## Fonctionnalités

- **Accès direct à la fiche** : clic sur une entrée pour rouvrir la fiche
- **Tri** : tri par date (plus récent en premier ou plus ancien en premier)
- **Filtrage** : filtrage par type d'établissement
- **Recherche** : recherche textuelle dans l'historique

## Règles

- L'historique est **personnel** : chaque utilisateur voit uniquement son propre historique
- L'historique est **automatique** : chaque consultation d'une fiche est enregistrée
- L'historique est **persistant** : les entrées sont conservées jusqu'à ce que l'utilisateur les supprime (si fonctionnalité disponible)

## Cas limites

- Si aucune fiche n'a été consultée : message "Aucun historique disponible"
- Si l'historique est vide : invitation à consulter des fiches pour commencer à construire l'historique
