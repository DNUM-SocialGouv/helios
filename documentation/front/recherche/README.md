# Recherche

La fonctionnalité de recherche permet de retrouver rapidement une entité juridique ou un établissement territorial (sanitaire ou médico-social) à partir de différents critères.

## Types de recherche

- **[Recherche simple](recherche-simple.md)** : recherche rapide par nom, raison sociale ou numéro FINESS.
- **[Recherche avancée](recherche-avancee.md)** : recherche avec filtres multiples (géographie, type, catégorie, etc.).

## Fonctionnalités communes

- Affichage des résultats en vignettes ou en tableau.
- Historique des recherches (enregistré automatiquement).
- Ajout aux favoris depuis les résultats.
- Ajout à une liste depuis les résultats.
- Accès direct à la fiche détaillée depuis les résultats.
