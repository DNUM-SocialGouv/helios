# Création d'une liste

## Pages où créer une liste

[Capture écran : création de liste depuis les résultats de recherche]

Une liste peut être créée depuis plusieurs pages :

1. **Page "Mes listes"** : accès direct depuis le menu
2. **Résultats de recherche** : après avoir sélectionné des établissements
3. **Page de comparaison** : après avoir sélectionné des établissements à comparer
4. **Fiche établissement** : depuis le bouton "Ajouter à une liste"

## Étapes de création

### Depuis la page "Mes listes"

[Capture écran : formulaire de création de liste]

1. Accéder à "Mes listes" depuis le menu
2. Cliquer sur le bouton "Créer une liste"
3. Saisir le nom de la liste
4. Optionnellement, ajouter des établissements immédiatement (via recherche ou import)
5. Valider la création

### Depuis les résultats de recherche

1. Effectuer une recherche (simple ou avancée)
2. Sélectionner un ou plusieurs établissements (cases à cocher)
3. Cliquer sur le bouton "Actions" puis "Ajouter à une liste"
4. Choisir "Créer une nouvelle liste"
5. Saisir le nom de la liste
6. Valider

### Depuis la page de comparaison

1. Ajouter des établissements à la comparaison
2. Cliquer sur "Actions" puis "Ajouter à une liste"
3. Choisir "Créer une nouvelle liste"
4. Saisir le nom de la liste
5. Valider

### Depuis une fiche établissement

1. Ouvrir la fiche d'un établissement
2. Cliquer sur le bouton "Ajouter à une liste"
3. Choisir "Créer une nouvelle liste"
4. Saisir le nom de la liste
5. Valider

## Règles

- Le nom de la liste est obligatoire
- Le nom doit être unique pour l'utilisateur
- La liste est créée immédiatement et peut être modifiée ensuite
- La liste est privée par défaut (non partagée)
