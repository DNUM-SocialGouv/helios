# Ajout de plusieurs établissements à une liste

## Pages où ajouter plusieurs établissements

L'ajout groupé de plusieurs établissements est possible depuis :

1. **Résultats de recherche** (simple ou avancée)
2. **Page de comparaison**
3. **Page "Mes listes"** (via recherche)

## Étapes d'ajout groupé

### Depuis les résultats de recherche

[Capture écran : sélection multiple dans résultats]

1. Effectuer une recherche
2. Sélectionner plusieurs établissements en cochant les cases à cocher
3. Cliquer sur le bouton "Actions"
4. Choisir "Ajouter la sélection à une liste"
5. Sélectionner la liste de destination (ou créer une nouvelle liste)
6. Valider

**Limite** : tous les établissements de la page actuelle peuvent être sélectionnés. Pour sélectionner sur plusieurs pages, répéter l'opération page par page.

### Depuis la page de comparaison

1. Ajouter plusieurs établissements à la comparaison
2. Sélectionner les établissements souhaités (cases à cocher dans le tableau)
3. Cliquer sur "Actions" puis "Ajouter la sélection à une liste"
4. Choisir la liste de destination
5. Valider

### Depuis la page "Mes listes"

1. Ouvrir une liste
2. Utiliser la recherche pour trouver plusieurs établissements
3. Sélectionner les établissements trouvés
4. Ajouter à la liste

## Règles

- La sélection peut contenir des établissements de types différents (sanitaire, médico-social, entité juridique)
- L'ajout est effectué en une seule opération
- Un message de confirmation indique le nombre d'établissements ajoutés
- Les doublons ne sont pas vérifiés automatiquement
