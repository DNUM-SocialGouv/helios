# Gestion d'une liste

## Modifier le titre d'une liste

[Capture écran : modification du titre]

1. Accéder à "Mes listes" depuis le menu
2. Repérer la liste à modifier
3. Cliquer sur l'icône "Modifier" (crayon) à côté du nom de la liste
4. Saisir le nouveau nom
5. Valider

**Règles** :
- Le nouveau nom doit être unique pour l'utilisateur
- La modification est immédiate

## Supprimer une liste

[Capture écran : suppression de liste]

1. Accéder à "Mes listes" depuis le menu
2. Repérer la liste à supprimer
3. Cliquer sur l'icône "Supprimer" (poubelle)
4. Confirmer la suppression

**Règles** :
- La suppression est définitive et irréversible
- Tous les établissements de la liste sont également supprimés
- Un message de confirmation est demandé

## Fonctionnalités disponibles depuis le bouton Actions

[Capture écran : menu Actions sur une liste]

Depuis la vue détaillée d'une liste, le bouton "Actions" permet :

- **Exporter la liste** : export Excel de tous les établissements de la liste avec leurs indicateurs
- **Comparer la liste** : lancer une comparaison avec tous les établissements de la liste
- **Partager la liste** : partager la liste avec d'autres utilisateurs (si rôle habilité)
- **Importer des établissements** : ajouter des établissements depuis un fichier FINESS
- **Supprimer la liste** : supprimer définitivement la liste

## Partage d'une liste

[Capture écran : partage de liste]

Le partage est disponible uniquement pour les rôles ADMIN_NAT et ADMIN_REG :

1. Ouvrir la liste à partager
2. Cliquer sur "Actions" puis "Partager"
3. Choisir le niveau de partage :
   - Régional : visible par tous les utilisateurs de la région
   - National : visible par tous les utilisateurs
4. Valider

**Règles** :
- Une liste partagée reste modifiable par son créateur
- Les autres utilisateurs peuvent consulter et utiliser la liste mais ne peuvent pas la modifier
- Le partage peut être révoqué à tout moment
