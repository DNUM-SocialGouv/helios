# Suppression de plusieurs établissements d'une liste

## Pages où supprimer plusieurs établissements

La suppression groupée est possible depuis :

1. **Page "Mes listes"** : vue détaillée d'une liste
2. **Page de comparaison** : si la liste est chargée

## Étapes de suppression groupée

### Depuis la page "Mes listes"

[Capture écran : sélection multiple pour suppression]

1. Accéder à "Mes listes" depuis le menu
2. Ouvrir la liste concernée
3. Sélectionner plusieurs établissements en cochant les cases à cocher
4. Cliquer sur le bouton "Actions"
5. Choisir "Supprimer la sélection"
6. Confirmer la suppression

### Depuis la page de comparaison

1. Charger une liste dans la comparaison
2. Sélectionner plusieurs établissements dans le tableau
3. Cliquer sur "Actions" puis "Supprimer la sélection"
4. Confirmer

## Règles

- La suppression groupée est définitive et immédiate
- Un message de confirmation indique le nombre d'établissements à supprimer
- Après suppression, la liste est automatiquement mise à jour
- Si tous les établissements sont supprimés, la liste devient vide mais n'est pas supprimée automatiquement
