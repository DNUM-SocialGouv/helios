# Ajout d'un établissement à une liste

## Pages où ajouter un établissement

Un établissement peut être ajouté à une liste depuis :

1. **Résultats de recherche** (simple ou avancée)
2. **Page de comparaison**
3. **Fiche établissement** (sanitaire, médico-social ou entité juridique)
4. **Page "Mes listes"** (via recherche dans la liste)

## Étapes d'ajout

### Depuis les résultats de recherche

[Capture écran : ajout depuis résultats]

1. Effectuer une recherche
2. Cliquer sur l'icône "Ajouter à une liste" sur un résultat (ou sélectionner plusieurs résultats)
3. Si plusieurs résultats sélectionnés : cliquer sur "Actions" puis "Ajouter à une liste"
4. Choisir la liste de destination dans le menu déroulant
5. Valider

### Depuis une fiche établissement

[Capture écran : bouton ajouter à une liste sur fiche]

1. Ouvrir la fiche d'un établissement
2. Cliquer sur le bouton "Ajouter à une liste"
3. Choisir la liste de destination
4. Valider

### Depuis la page de comparaison

1. Ajouter des établissements à la comparaison
2. Sélectionner les établissements à ajouter (cases à cocher)
3. Cliquer sur "Actions" puis "Ajouter à une liste"
4. Choisir la liste de destination
5. Valider

### Depuis la page "Mes listes"

1. Ouvrir une liste existante
2. Utiliser la fonction de recherche pour trouver des établissements
3. Ajouter les établissements trouvés à la liste

## Règles

- Un établissement peut être présent plusieurs fois dans une liste (pas de vérification de doublon automatique)
- L'ajout est immédiat et enregistré automatiquement
- Un message de confirmation s'affiche après l'ajout
