# Suppression d'un établissement d'une liste

## Pages où supprimer un établissement

Un établissement peut être supprimé d'une liste depuis :

1. **Page "Mes listes"** : vue détaillée d'une liste
2. **Page de comparaison** : si la liste est chargée dans la comparaison

## Étapes de suppression

### Depuis la page "Mes listes"

[Capture écran : suppression depuis liste]

1. Accéder à "Mes listes" depuis le menu
2. Ouvrir la liste concernée
3. Repérer l'établissement à supprimer dans le tableau
4. Cliquer sur l'icône "Supprimer" (poubelle) sur la ligne de l'établissement
5. Confirmer la suppression

### Depuis la page de comparaison

1. Charger une liste dans la comparaison
2. Dans le tableau, repérer l'établissement à supprimer
3. Cliquer sur l'icône "Supprimer" sur la ligne
4. Confirmer la suppression

## Règles

- La suppression est définitive et immédiate
- Un message de confirmation est demandé avant suppression
- Après suppression, la liste est automatiquement mise à jour
