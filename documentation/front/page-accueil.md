# Page d'accueil

## Bandeau retour page d'accueil

[Capture écran : bandeau avec logo et lien retour accueil]

Le bandeau supérieur de chaque page contient :

- Le logo Helios (cliquable pour retourner à la page d'accueil).
- Le menu de navigation principal.
- Le menu utilisateur (en haut à droite).

En cliquant sur le logo ou sur "Accueil", l'utilisateur est redirigé vers la page d'accueil principale.

## Bandeau d'information

[Capture écran : bandeau d'information]

Un bandeau d'information peut être affiché en haut de la page d'accueil pour communiquer :

- Des messages importants (maintenance, nouvelles fonctionnalités).
- Des alertes système.
- Des informations générales.

Ce bandeau est paramétrable par les administrateurs et peut être masqué par l'utilisateur.

## Contenu de la page d'accueil

[Capture écran : page d'accueil complète]

La page d'accueil présente :

- Le formulaire de connexion (si non connecté) ou un tableau de bord (si connecté).
- Les liens vers les principales fonctionnalités :
  - Recherche
  - Mes listes
  - Comparaison
  - Aide
- Les informations de contact et liens utiles (mentions légales, gestion des cookies).
