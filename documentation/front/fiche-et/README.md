# Fiche établissement / Entité juridique

Les fiches détaillées permettent d'accéder à l'ensemble des indicateurs et informations d'une entité juridique ou d'un établissement territorial.

## Types de fiches

- **[Fiche entité juridique](fiche-ej.md)** : vue consolidée d'une entité juridique avec ses établissements rattachés.
- **[Fiche établissement sanitaire](fiche-et-san.md)** : vue détaillée d'un établissement de santé.
- **[Fiche établissement médico-social](fiche-et-ms.md)** : vue détaillée d'un établissement médico-social.

## Structure commune des fiches

Toutes les fiches sont organisées en blocs thématiques :

- **Identité** : informations générales (raison sociale, FINESS, adresse, etc.)
- **Activité** : indicateurs d'activité selon le type d'établissement
- **Budget & Finances** : données financières et budgétaires
- **Ressources Humaines** : effectifs, absentéisme, intérim
- **Autorisations** : autorisations sanitaires ou médico-sociales
- **Qualité** : réclamations, événements indésirables, inspections

Chaque bloc indique :
- L'année de référence des données
- La date de dernière mise à jour
- Un message si les données sont indisponibles ou obsolètes

## Navigation parent/enfant

Depuis une fiche entité juridique, il est possible de naviguer vers les établissements rattachés, et inversement depuis un établissement vers son entité juridique parente.
