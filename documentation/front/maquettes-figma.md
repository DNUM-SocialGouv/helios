## Utilisation des maquettes

Les maquettes Figma permettent de :

- **Visualiser les interfaces** avant leur implémentation
- **Comprendre les interactions** et les parcours utilisateur
- **Valider les choix de design** avec les équipes produit et métier
- **Servir de référence** pour le développement frontend

## Notes importantes

- Les maquettes sont régulièrement mises à jour pour refléter l'évolution de l'interface
- Certaines fonctionnalités peuvent différer légèrement entre les maquettes et l'application en production
- Pour toute question sur les maquettes, contacter l'équipe design
