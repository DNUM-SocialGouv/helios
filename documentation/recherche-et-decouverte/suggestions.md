# Suggestions et catégories FINESS

Le système de suggestions et de catégories FINESS facilite la recherche en guidant l'utilisateur dans la sélection des critères et en proposant des catégories d'établissements pertinentes.

[Capture écran : Interface de recherche avancée avec suggestions de catégories]

## Catégories FINESS

### Taxonomie harmonisée

Les catégories FINESS sont organisées selon une taxonomie officielle qui classe les établissements par domaine d'activité. Chaque catégorie possède :
- **Un code** : identifiant unique de la catégorie.
- **Un libellé complet** : nom officiel de la catégorie.
- **Un libellé court** : version abrégée pour l'affichage.
- **Un domaine** : regroupement par grande famille (sanitaire, médico-social, etc.).

### Recherche dans les catégories

Lors de la sélection des catégories dans la recherche avancée :
- **Recherche textuelle** : l'utilisateur peut saisir un terme pour filtrer la liste des catégories disponibles.
- **Correspondance flexible** : la recherche s'effectue sur le code, le libellé complet et le libellé court, en ignorant les accents et la casse.
- **Affichage filtré** : seules les catégories correspondant au terme saisi sont affichées, facilitant la sélection dans une liste qui peut être longue.

[Capture écran : Champ de recherche dans les catégories avec résultats filtrés]

### Sélection multiple

- **Plusieurs catégories** : l'utilisateur peut sélectionner plusieurs catégories simultanément pour élargir sa recherche.
- **Indicateur visuel** : le nombre de catégories sélectionnées est affiché pour informer l'utilisateur.
- **Désélection** : possibilité de retirer une catégorie de la sélection à tout moment.

[Capture écran : Liste des catégories sélectionnées avec possibilité de désélection]

## Suggestions intelligentes

### Propositions contextuelles

Le système peut proposer des suggestions basées sur :
- **Historique de recherche** : les catégories fréquemment utilisées par l'utilisateur peuvent être mises en avant.
- **Contexte régional** : suggestions adaptées aux établissements présents dans la région de l'utilisateur.
- **Recherches similaires** : propositions basées sur les recherches effectuées par d'autres utilisateurs dans des contextes similaires.

### Affichage des suggestions

- **Intégration discrète** : les suggestions apparaissent de manière non intrusive, sans perturber le flux de recherche.
- **Action rapide** : un clic sur une suggestion applique directement le critère correspondant.

[Capture écran : Suggestions affichées sous le champ de recherche]

## Règles de fonctionnement

1. **Cohérence des catégories** : les catégories proposées correspondent toujours aux établissements réellement présents dans la base de données.
2. **Mise à jour automatique** : la liste des catégories est synchronisée avec les référentiels officiels pour garantir l'exactitude.
3. **Performance** : le filtrage des catégories est optimisé pour rester réactif même avec un grand nombre de catégories disponibles.

## Cas limites

- **Aucune catégorie correspondante** : si la recherche textuelle ne trouve aucune catégorie, un message indique qu'aucun résultat n'a été trouvé et suggère d'essayer un autre terme.
- **Catégories obsolètes** : les catégories qui ne sont plus utilisées dans les référentiels officiels sont progressivement retirées, mais restent accessibles pour les recherches historiques.
